# Android Studio Lite Settings

### Tool Windows
| Action | Shortcut |
|---|---|
| Project| ⌘1 |
| Build  | ⌘2  |
| Run  | ⌘3  |
| Debug  | ⌘4  |
| Logcat  | ⌘5  |
| Terminal  | ⌘6  |
| Git | ⌘7  |
| Device Manager | ⌘8  |

### Compilation and Running
| Action | Shortcut |
|---|---|
| Run| ^R |
| Debug  | ^D  |
| Attach Debugger  | ⌥D  |
| Make  | ^M  |
| Clean  | ^C  |
| Sync  | ^S  |

### Find and Search
| Action | Shortcut |
|---|---|
| Search all actions| ⌥A |
| Search class structure  | ⌥S  |
| Search recent files  | ⌥R  |

### Others
| Action | Shortcut |
|---|---|
| Split vertically| ^⌥V |

### Live Templates
| Template | Description |
|---|---|
| cfun  | create composable function  |
| pfun  | create composable preview function  |
| sfun| create suspend function |
| tfun| create test function |
